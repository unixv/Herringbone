# Herringbone OpenSOC
Bite sized security tool components for you to stack however you want!

## Log Ingestion
Herringbone Log Ingestion is an advanced SIEM (Security Information and Event Management) tool designed to offer unparalleled flexibility and scalability through a containerized architecture. Each component of the system is modular, allowing users to seamlessly stack and customize elements to suit their specific security requirements.

The tool's log receivers support both TCP and UDP protocols, ensuring robust and reliable log ingestion. Herringbone Log Ingestion is further enhanced with sophisticated machine learning capabilities, which perform log type and parsing predictions. These components continuously learn and improve over time, adapting to new data and evolving threats.

This modular design not only simplifies deployment and management but also enables easy expansion and integration of additional features as needed. Ideal for organizations of all sizes, Herringbone Log Ingestion provides a powerful, adaptable solution for comprehensive security monitoring and incident response.
